# PaResume

PaResume is an **AI-powered Resume Analyzer** built with a modern React + Vite stack. It helps users upload resumes, analyze their content, and get ATS-friendly suggestions for optimization.

---

## Features

- Upload resumes and analyze ATS compatibility
- View keyword and skill suggestions
- Interactive and fast UI with React + Vite
- State management using Zustand
- Smooth navigation with React Router v2

---

## Tech Stack

**Frontend:**

- [React](https://react.dev/) – UI library
- [Vite](https://vitejs.dev/) – Fast build tool
- [TypeScript](https://www.typescriptlang.org/) – Type-safe JavaScript
- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework
- [Zustand](https://github.com/pmndrs/zustand) – Lightweight state management
- [React Router v2](https://reactrouter.com/) – Client-side routing
- [Puter.js](https://puter.js.org/) – For AI/utility integration
